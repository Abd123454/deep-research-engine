# DOCX Skill

## When to use
- Formal documents, letters, reports for Word users
- When the user asks for a .docx file

## Format rules
- Use Heading 1 for title, Heading 2 for sections
- Body text in normal paragraphs
- Tables for structured data
- Page breaks between major sections

## Constraints
- Max 1000 pages
- Images must be base64 or URL
- No macros
- Font defaults to Calibri 11pt
