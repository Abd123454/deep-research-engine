# Markdown Skill

## When to use
- Reports, essays, articles, guides
- Standalone written content >20 lines

## Format rules
- Use # for title, ## for sections, ### for subsections
- Code blocks with language tags
- Inline code with backticks
- Links as [text](url)

## Style
- Write in prose, not bullet lists
- Use lists only when content is genuinely multifaceted
- Avoid excessive bold emphasis
- Keep paragraphs 3-5 sentences

## Avoid
- Bullet lists for prose
- Headers every paragraph
- Excessive formatting
