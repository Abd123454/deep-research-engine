# XLSX Skill

## When to use
- Data tables, financial models, calculations
- When the user asks for a spreadsheet

## Format rules
- Headers in row 1, bold
- Data starts row 2
- Use formulas for calculations (=SUM, =AVERAGE)
- One sheet per data category

## Constraints
- Max 1M rows
- No macros
- Numbers right-aligned, text left-aligned
