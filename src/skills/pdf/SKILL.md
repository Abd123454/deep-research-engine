# PDF Skill

## When to use
- Formal reports, invoices, certificates
- When the user asks for a PDF

## Format rules
- A4 page size
- 1-inch margins
- 12pt body font, 16pt headings
- Page numbers in footer

## Constraints
- No interactive elements
- No forms
- Text is not editable after generation
