# PPTX Skill

## When to use
- Presentations, slideshows, pitch decks
- When the user asks for slides

## Format rules
- One topic per slide
- Title + 3-5 bullet points max
- Use ## for slide titles, content as bullet list
- Keep text concise (audience reads, not studies)

## Constraints
- Max 100 slides
- No animations
- 16:9 widescreen format
- Bullet points must be <100 chars each
