Features

Everything you need for AI-powered research and automation, in one self-hosted platform.

🔬

Deep Research

6-stage pipeline: plan → decompose → search → gap analysis → round 2 → synthesize. Citation verification checks every URL.

🎯

Source Quality Scoring

Tier-based ranking: academic/government (tier 1) → industry (tier 2) → general web (tier 3). Weak sources dropped.

🌐

JS-Rendered Reading

Playwright headless browser fallback for SPA sites (React/Vue/Angular) that return empty HTML to direct fetch.

🤖

Agentic Chat

ReAct (Reason + Act) loop with native tool calling. Skills system: Default, Coder, Researcher, Writer, Data Analyst.

🐝

Agent Swarm

Multi-agent collaboration: orchestrator breaks tasks, specialists run in parallel, synthesizer combines outputs.

⌨️

Code Sandbox

JavaScript (vm) + Python (subprocess) + Docker (isolated). Verifier loop: code fails → agent fixes → retries.

👁️

Vision

4-provider fallback: OpenAI GPT-4o → Anthropic Claude 3.5 → NVIDIA Llama 3.2 → Tesseract OCR.

🎙️

Voice

Text-to-speech (TTS) + speech-to-text (ASR) with multi-provider support.

📄

File Generation

Export to PDF, DOCX, PPTX, XLSX. Generate documents from research reports.

🎨

3-Column Layout

Sidebar (conversations) + Chat (main) + Artifacts (interactive HTML/React/SVG/code results).

🧠

5-Layer Memory

Conversation history, long-term semantic memory, research history, documents, user preferences.

🔄

Multi-Provider LLM

NVIDIA (6-model chain) → OpenAI → Anthropic → Ollama. If NVIDIA fails, OpenAI takes over automatically.

⏹️

Real Cancellation

AbortController cancels in-flight HTTP requests immediately. No wasted budget when you click Stop.

📱

PWA

Installable, offline indicator, service worker. Works like a native app on mobile.

🧩

Browser Extension

Chrome/Firefox (Manifest V3): capture any page and research it with one click.

🖥️

Desktop App

Electron wrapper: system tray, native menu, keyboard shortcuts. Native desktop experience.

Quick Start

Get running in under 2 minutes.

# Clone the repository git clone https://github.com/Abd123454/deep-research-engine.git cd deep-research-engine # Install dependencies bun install # Configure environment cp .env.example .env # Edit .env: add NVIDIA_API_KEY (free at https://build.nvidia.com/) # Start the dev server bun run dev # Open http://localhost:3000 # Run the evaluation suite bun run eval

Known Limitations

Honest about what's not perfect yet.

⚠️ Honest Gaps

  • research-engine.ts test coverage: ~23% (targeting 60%)
  • No mobile app (planned for future release)
  • No multi-user isolation (single-user by default)
  • Rate limiter uses in-memory fallback without Redis
  • Docker sandbox requires Docker installed (falls back to vm)
  • Playwright adds ~150MB (optional, for JS-rendered pages)
