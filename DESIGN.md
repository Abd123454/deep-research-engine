# Quaesitor Design System — "The Investigator's Journal"

## Philosophy

Quaesitor (Latin: "the seeker, the investigator") is designed to feel
like a researcher's well-worn journal — warm, trustworthy, and built
for deep inquiry. The aesthetic draws from:

- **Aged paper and leather bindings** — warm ambers, saddle browns, sepia inks
- **Academic publishing** — footnotes, citation lists, source tier ratings
- **Cartographic exploration** — compass rose logo, depth markers, progress bars

This is an independent design identity. While it shares the broader
"warm editorial" aesthetic with other products, all color values,
typography choices, and component dimensions are original to Quaesitor.

## Design Principles

1. **Warmth over coolness** — sepia tones throughout, never pure gray
2. **Reading-first** — Newsreader font at 18px, 1.7 line-height for comfort
3. **Investigation metaphor** — depth indicators, citation footnotes, source tiers
4. **Editorial calm** — no shadows, no gradients, borders only for elevation
5. **Independent identity** — inspired by journal aesthetics, not copying any product

## Typography

- **Fraunces** (display headlines) — soft-serif with personality, inspired by Windsor/Souvenir
- **Newsreader** (body) — designed for on-screen reading, commissioned by Google Fonts
- **DM Sans** (UI chrome) — geometric, clear, distinct from Inter
- **JetBrains Mono** (code) — excellent readability

## Signature Elements

- **Depth Indicator** — 3-dot camera-lens style (quick/standard/deep)
- **Citation Footnotes** — academic [1] style with reference list
- **Source Tier Badges** — ★★★ academic / ★★☆ industry / ★☆☆ general
- **Investigation Progress** — ◈ diamond + progress bar (not spinner)
- **Compass Rose Logo** — 4-point star, symbol of exploration

## Component Dimensions

- Sidebar: 280px (wider for investigation context)
- Composer: rounded-3xl (24px), warm border
- User bubble: rounded-3xl + bottom-right sharp (24px/8px), max-w-75%
- Cards: rounded-[20px]
- Body: 18px, line-height 1.7

## Anti-patterns (forbidden)

- No box-shadow (elevation via surface tone + borders)
- No backdrop-blur
- No bg-gradient
- No pure black (#000) or pure white (#fff)
